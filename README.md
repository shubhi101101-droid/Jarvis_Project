# 🎙️ Jarvis Voice Assistant

A simple AI-powered voice assistant built with Python that can open websites, play music, and respond to voice commands — just like your own mini **Jarvis**.

---

## 🚀 Features

* 🎤 Voice command recognition using SpeechRecognition
* 🔊 Text-to-speech responses with pyttsx3
* 🌐 Open popular websites instantly
* 🎵 Play songs from a custom music library
* ⚡ Wake-word activation using **"Jarvis"**
* 🧠 Lightweight and beginner-friendly project

---

## 🛠️ Tech Stack

* Python
* SpeechRecognition
* pyttsx3
* Webbrowser
* PyAudio

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/jarvis-voice-assistant.git
cd jarvis-voice-assistant
```

Install required libraries:

```bash
pip install SpeechRecognition pyttsx3 pyaudio
```

---

## ▶️ Run the Project

```bash
python main.py
```
